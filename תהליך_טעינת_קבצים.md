# תהליך טעינת קבצי CSV למונגו

## סקירה כללית

המערכת מקבלת קבצי CSV שמגיעים ממשרד החינוך, ממירה אותם למסמכי MongoDB עם המרות טיפוסים אוטומטיות, ושומרת אותם בקולקשנים המתאימים.

---

## ממשק המשתמש — דף `/upload`

### בחירת קבצים
- בוחרים **תיקייה שלמה** (באמצעות `webkitdirectory` attribute) — כל הקבצים בתיקייה נטענים
- או בוחרים **קבצים בודדים** (multiple file select)

### אפשרות טיפול בכפולות
לפני הייבוא, המשתמש בוחר:
- **דלג** (ברירת מחדל) — אם קובץ כבר קיים במונגו (לפי `source_file`) → דלג עליו
- **דרוס** — מחק את כל הרשומות של הקובץ הקיים → ייבא מחדש

### תצוגת תוצאות
לאחר הייבוא, מוצגת טבלת סיכום לכל קובץ עם סטטוס:
- ✅ `imported` — קובץ חדש, נייבא בהצלחה
- ♻️ `replaced` — קובץ קיים, נמחק ונוייבא מחדש
- ⏭️ `skipped` — קובץ קיים, דולג
- ❌ `error` — שגיאה (שם קובץ לא תקין, קובץ ריק)

## פורמט שם הקובץ

**דפוס חובה**: `{מספר}_{חודש}_{שנה}{FILETYPE}.csv`

```
דוגמאות תקינות:
12345_03_2025CHESHBONIT.csv
98765_11_2024MUCARIM.csv
11111_1_2025YADANIIM.csv

Regex לבדיקה: /^(\d+)_(\d{1,2})_(\d{4})([A-Z0-9]+)\.csv$/i
```

**קבוצות ה-match**:
- `match[1]` — מספר מזהה (מוסד/ריצה)
- `match[2]` — חודש (1-12)
- `match[3]` — שנה (4 ספרות)
- `match[4]` — **FILETYPE** → קובע לאיזה קולקשן הנתונים נשמרים

**מיפוי FILETYPE → קולקשן** (מ-`config.ts`):
```
CHESHBONIT → CHESHBONIT
MUCARIM    → MUCARIM
SHARATIM   → SHARATIM
YADANIIM   → YADANIIM
GY         → GY
HASAOT     → HASAOT
HASNET     → HASNET
HASMASLULIM → HASMASLULIM
MISROT     → MISROT
MISROTGY   → MISROTGY
MOADON     → MOADON
MUTAVIM    → MUTAVIM
SHEFI      → SHEFI
```

## לוגיקת הייבוא (`POST /api/import`)

### קלט
`multipart/form-data` עם:
- `files[]` — קבצי CSV
- `duplicateStrategy` — `'skip'` או `'replace'`

### עיבוד כל קובץ

#### שלב 1: פרסור שם הקובץ
```javascript
const filename = rawFilename.split('/').pop()?.split('\\').pop() || rawFilename;
const match = filename.match(/^(\d+)_(\d{1,2})_(\d{4})([A-Z0-9]+)\.csv$/i);
if (!match) {
  results.push({ filename, status: 'error', error: 'שם קובץ לא תואם לפורמט' });
  continue;
}
const fileType = match[4].toUpperCase();
const collectionName = CONFIG.collections[fileType] || fileType;
```

#### שלב 2: בדיקת כפולות
```javascript
const existingCount = await db.collection(collectionName)
                               .countDocuments({ source_file: filename });
if (existingCount > 0 && duplicateStrategy === 'skip') {
  results.push({ filename, collection: collectionName, rows: existingCount, status: 'skipped' });
  continue;  // לא קוראים את ה-CSV בכלל — חיסכון בזמן!
}
```

#### שלב 3: קריאת ה-CSV
```javascript
const buffer = Buffer.from(await file.arrayBuffer());
const records = [];

await new Promise((resolve, reject) => {
  const stream = Readable.from(buffer.toString('utf-8'));
  stream
    .pipe(csvParser({
      mapHeaders: ({ header }) => normalizeColumnName(header),
      quote: '\x00',    // מבטל את עיבוד quotes (קבצים לא עוטפים שדות)
      escape: '\x00'    // מבטל escape characters
    }))
    .on('data', (data) => {
      // המרות שדות
      for (const key in data) {
        if (key === 'חודש_חישוב' || key === 'חודש_תחולה') {
          data[key] = parseDateString(String(data[key])) || data[key];
        } else {
          data[key] = formatValue(data[key]);
        }
      }
      // metadata
      data.source_file = filename;
      data.import_timestamp = new Date();
      data.run_id = runId;
      data.file_type = fileType;
      records.push(data);
    })
    .on('end', resolve)
    .on('error', reject);
});
```

#### שלב 4: שמירה ב-MongoDB
```javascript
if (existingCount > 0 && duplicateStrategy === 'replace') {
  await db.collection(collectionName).deleteMany({ source_file: filename });
  await db.collection(collectionName).insertMany(records);
  results.push({ ..., status: 'replaced' });
} else {
  await db.collection(collectionName).insertMany(records);
  results.push({ ..., status: 'imported' });
}
```

## פונקציות עזר

### `normalizeColumnName(col: string): string`
מנקה שמות עמודות מה-CSV:
```javascript
col.replace(/^\uFEFF/, '')   // הסרת BOM (Byte Order Mark) מתחילת הקובץ
   .trim()                   // הסרת רווחים מהצדדים
   .replace(/\s+/g, '_')     // המרת רווחים לקו תחתון
```

**חשוב**: קבצי CSV בעברית מ-Excel כוללים לרוב BOM (`\uFEFF`) בתחילת הקובץ — בלי הסרתו, שם העמודה הראשונה יכיל תו בלתי נראה שישבש מיפויים.

### `parseDateString(dateStr: string): Date | null`
ממיר תאריך בפורמט `MM/YYYY` (כמו `03/2025`) לאובייקט Date ב-UTC:
```javascript
const parts = dateStr.trim().split('/');
if (parts.length === 2) {
  const p1 = parseInt(parts[0], 10);  // חודש
  const p2 = parseInt(parts[1], 10);  // שנה
  if (p1 <= 12 && p2 > 1900) {
    return new Date(Date.UTC(p2, p1 - 1, 1));  // יום ראשון בחודש, UTC
  }
}
return null;
```

**מדוע UTC**: למניעת שגיאות timezone שיכולות לגרום לתאריך "לזוז" יום אחד אחורה.

### `formatValue(val: unknown): string | number | null`
ממיר ערכי שדה מ-string:
```javascript
if (typeof val !== 'string') return null;
const trimmed = val.trim();
if (trimmed === '') return null;
const num = Number(trimmed);
if (!isNaN(num)) return num;   // ערך מספרי → number
return trimmed;                 // טקסט → string
```

## הזרמת נתונים — תרשים

```
קבצי CSV מדיסק
      ↓
שליחה ל-POST /api/import (multipart/form-data)
      ↓
לכל קובץ:
  ├── בדיקת שם קובץ (regex)
  │     ↓ לא תואם → error
  ├── בדיקת כפולות במונגו
  │     ↓ קיים + skip → skipped (ללא קריאת CSV)
  ├── קריאת CSV עם csv-parser
  │     ├── normalizeColumnName (BOM + underscore)
  │     ├── parseDateString (חודש_חישוב, חודש_תחולה)
  │     └── formatValue (string → number/string/null)
  ├── הוספת metadata (source_file, import_timestamp, run_id, file_type)
  └── שמירה ב-MongoDB
        ├── קיים + replace → deleteMany + insertMany
        └── חדש → insertMany
```

## דגשים ושגיאות נפוצות

| בעיה | סיבה | פתרון |
|------|------|--------|
| עמודה ראשונה עם שם שגוי | BOM בקובץ | `col.replace(/^\uFEFF/, '')` |
| תאריך לא מומר ל-Date | פורמט שגוי (לא MM/YYYY) | בדוק שהקובץ ממשרד החינוך |
| שורות ריקות בסוף | שורות CSV ריקות | `formatValue` מחזיר null לשדות ריקים |
| שגיאת duplicate key | אין unique index, אבל בדיקת `source_file` נכשלה | בדוק את הלוגיקה של `duplicateStrategy` |
| קבצים לא נטענים | שם קובץ לא תואם regex | בדוק את פורמט `{מספר}_{חודש}_{שנה}{FILETYPE}.csv` |
